# Object Detection Model - YOLOv8 Nano

A lightweight object detection project built using **YOLOv8 Nano**, designed for fast inference and efficient deployment on resource-constrained devices. This repository demonstrates how to train, test, and run object detection using the YOLOv8 Nano architecture.

---

## 📌 Features
- Uses **YOLOv8 Nano** for real-time object detection.
- Lightweight model (`yolov8n.pt`) suitable for edge devices.
- Example scripts for training and inference.
- Easy to extend for custom datasets.

---

## 📂 Repository Structure
├── demo.py              # Script to run demo inference

├── objectdetection.py   # Core object detection pipeline

├── yolov8n.pt           # Pre-trained YOLOv8 Nano weights

├── .vscode/             # VSCode configuration files

└── README.md            # Project documentation
